# B-Tree Implementation

## Описание

Реализация **B-дерева (B-Tree)** на языке C.

B-дерево — это сбалансированное дерево поиска, широко используемое в базах данных и файловых системах для хранения больших объёмов данных на диске. В отличие от бинарных деревьев, узел B-дерева может содержать множество ключей, что минимизирует количество обращений к диску.

## Свойства B-дерева

- Каждый узел содержит от `t-1` до `2t-1` ключей (кроме корня)
- Все листья находятся на одном уровне
- Ключи в узле упорядочены
- Поиск, вставка и удаление выполняются за O(log n)
